# NgxAutohide

This library is used to automatically hide an Angular Component.

## ⚙️ Installation

```bash
npm install ngx-autohide --save
```
## 📖 Usage

### 1. By default timings is 5000ms(5s)*
```html
<div autoHide>
  No More Information!
</div>
```
### 2. Set timings = 10000ms(10s)*
```html
<div autoHide [timings]="10000">
  No More Information!
</div>
```
